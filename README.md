## 🚀 What Deal Engine Does

### Core Analysis Engine
- **Massing Studies** — Optimized FAR envelope, height limits, setback calculations
- **Yield Studies** — Unit counts, net leasable area, rental yield, income potential
- **Financial Modeling** — ROI, GDV, profit, development margin, cost per unit
- **DEAL Score™** — Instant 0–100 project viability rating
- **Risk Assessment** — Entitlement risk, NIMBY risk, market timing analysis

### Advanced Features
- **🎨 Concept Gallery** — 24 architectural styles with instant preview. Purchase complete design packages with 12 renders, presentation deck, and 3D model ($1,999)
- **📋 Deal Pipeline** — Track multiple deals, compare side-by-side
- **💰 Capital Stack Builder** — Model senior debt, mezzanine, preferred equity, sponsor equity
- **📈 Scenario Modeling** — Conservative, base, and aggressive projections
- **⚖️ Deal Comparison** — Side-by-side analysis of up to 3 deals
- **📚 Regulatory Intelligence** — Municipal code requirements, impact fees, entitlement pathways
- **🗺️ Competitive Pipeline** — What's being built nearby, absorption forecasts
- **🤝 Project Network** — Contact directory with one-tap call, SMS, and email
